```bash
        value=$(python3 -c "
import json
try:
    with open('$CONFIG_FILE') as f:
        cfg = json.load(f)
    # Navigate nested keys (e.g., 'task_routing.impl')
    keys = '$key'.split('.')
    v = cfg

            print(json.dumps(v))

            print(v)
except: pass
" 2>/dev/null)
```

Replace with:
```bash
        value=$(CONFIG_FILE="$CONFIG_FILE" KEY="$key" python3 <<'EOF' 2>/dev/null
import json
import os

try:
    config_file = os.environ['CONFIG_FILE']
    key = os.environ['KEY']

    with open(config_file) as f:
        cfg = json.load(f)
    # Navigate nested keys (e.g., 'task_routing.impl')
    keys = key.split('.')
    v = cfg
    for k in keys:
        if isinstance(v, dict):
            v = v.get(k)
        else:
            v = None
            break
    if v is not None:
        if isinstance(v, bool):
            print('true' if v else 'false')
        elif isinstance(v, (list, dict)):
            print(json.dumps(v))
        else:
            print(v)
except:
    pass
EOF
)
```

Key changes:
1. Replace `python3 -c "..."` with `python3 <<'EOF'` (quoted heredoc)
2. Pass `CONFIG_FILE` and `KEY` via environment variables
3. Add `import os`
4. Read variables with `os.environ['CONFIG_FILE']` and `os.environ['KEY']`
5. Use bare `except:` instead of `except: pass` on same line (Python style)
6. Close with `EOF` followed by `)` for subshell capture
