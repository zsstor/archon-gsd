- `test_failover_candidates_same_tier` — returns same-tier models
- `test_failover_candidates_no_downgrade` — excludes lower-tier models
- `test_failover_candidates_excludes_current` — excludes the failing model

**Q-04 Constraint tests:**
- `test_constraint_opus_only_waits` — opus-only tasks return empty failover list
- `test_constraint_flexible_allows_cascade` — flexible tasks return full failover list

**Q-05 Version pinning tests:**
- `test_version_pinning_allowed` — allowed version passes check
- `test_version_pinning_forbidden` — forbidden version fails check

**Q-06 State tests:**
- `test_quota_state_record` — records quota state to file
- `test_quota_state_is_limited` — returns 0 when model is limited
- `test_quota_state_ttl_expired` — returns 1 after TTL expires
- `test_quota_state_cleanup` — cleans up state files
